# MentalHealthApp

A lightweight **web & mobile platform** that helps users track their mental well-being and access self-help resources.

---

##  Core Use Case
Record moods, complete quick assessments, and view personal trendsall in one private, device-agnostic app.

---

##  Tech Stack
| Layer        | Technology |
|--------------|------------|
| Frontend     | **React Native** (mobile) 路 **React** + Vite (web) |
| Backend      | **Node.js** 路 **Express.js** |
| Database     | **MongoDB** (Mongoose ODM) |
| Auth & Security | JSON Web Tokens 路 BCrypt hashing |
| DevOps       | Docker 路 GitHub Actions 路 Render deployment |
| License      | MIT |
